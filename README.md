# Breast Cancer Classification

This project applies supervised learning techniques to the Breast Cancer dataset from sklearn.

## Models Implemented
- Logistic Regression
- Decision Tree
- Random Forest
- Support Vector Machine (SVM)
- k-Nearest Neighbors (k-NN)

## Results
- Logistic Regression: 97.37%
- SVM: 97.37%
- Random Forest: 96.49%
- Decision Tree: 94.74%
- k-NN: 94.74%

## Conclusion
SVM and Logistic Regression performed best, while Decision Tree and k-NN had the lowest accuracy.
